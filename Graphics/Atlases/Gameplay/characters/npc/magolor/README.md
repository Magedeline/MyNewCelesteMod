# Magolor Sprite Assets

This directory should contain the sprite images for NPC06_Magolor:

## Required Sprites:
- idle00.png through idle03.png - Idle animation frames
- wave04.png through wave07.png - Wave animation frames  
- point08.png through point11.png - Point animation frames
- fly12.png through fly15.png - Flying animation frames

## Sprite Specifications:
- Size: Approximately 24x32 pixels
- Style: Should match the overall art style of the mod
- Background: Transparent PNG format
- Character: Magolor - a round, friendly character with a jester-like appearance

## Animation Details:
- **Idle**: Gentle bobbing/breathing animation
- **Wave**: Friendly greeting wave gesture
- **Point**: Pointing gesture to indicate direction
- **Fly**: Wings/cape fluttering for flight motion

## Notes:
These sprites will be used in the CS06_Gondola cutscene where Magolor serves as a guide and companion during the gondola sequences.
