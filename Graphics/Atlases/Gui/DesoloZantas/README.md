# Desolo Zantas Title Screen Branding Assets

This folder contains custom branding assets for the Desolo Zantas mod's title screen.

## Required Files

### logo.png
- **Purpose**: The main logo displayed on the title screen when Desolo Zantas branding is enabled
- **Recommended Size**: Match the vanilla Celeste logo dimensions (approximately 960x540 pixels for best results)
- **Format**: PNG with transparency support
- **Location**: Place directly in this folder as `logo.png`

### title.png (Optional)
- **Purpose**: Title text/reflection displayed below the logo
- **Format**: PNG with transparency support
- **Location**: Place directly in this folder as `title.png`

## How to Use

1. Place your custom `logo.png` in this folder (`Graphics/Atlases/Gui/DesoloZantas/`)
2. Start the game and go to the title screen
3. Press the **Left Arrow** key or button to toggle between Celeste and Desolo Zantas branding
4. Confirm the swap - the game will restart to apply changes safely

## Settings

You can also enable/disable Desolo Zantas branding from the mod settings menu:
- Go to **Options** > **Mod Options** > **Desolo Zantas**
- Toggle **Use Desolo Zantas Branding**
- The change requires a game restart to take effect

## Notes

- The branding swap feature requires a game restart for safety
- If no custom logo is found, the system will fall back to the vanilla Celeste logo
- The left arrow on the title screen only appears when custom assets are available
